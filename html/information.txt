#Links and hyperlinks
Link refers to the web address (URL) of a resource, such as a website or file.  It is a plain, 
static reference to a location on the internet. For example, 
typing https://www.google.com into a browser is using a link. 
Hyperlink is a clickable element—often text, image, or button—embedded within a document or
 webpage that directs users to another resource when activated.
